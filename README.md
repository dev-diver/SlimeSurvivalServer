# Slime Servival Server

## Concept

- 멀티 플레이어가 먹이와 상대방을 잡아 먹으며 슬라임을 성장시켜 최후에 살아남는 게임

## 도전 요소

- 멀티 플레잉 게임을 처음으로 구현
- 저수준 websocket 라이브러리로 구현 후 프레임워크 도입
- 부드러운 동기화 기법 학습
- 성능 프로파일링 학습
- 여러 기법 적용 전후의 성능 비교
- 클라이언트 핵 제작과 방어

## 기술 스택

- 언어: go
